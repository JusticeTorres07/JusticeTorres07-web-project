# GreenTech Solutions Website / The Final

## Project Description

This website was created for GreenTech Solutions, a startup focused on environmental sustainability and technology. The purpose of the website is to inform small and medium-sized businesses about eco-friendly technology solutions, including energy-efficient devices, smart recycling systems, and carbon tracking software.

The website also provides resources about sustainability and includes a contact form for users who want to learn more about GreenTech Solutions.

## Pages Included

- The Final / Home Page
- About Us
- Products/Services
- Blog/Resources
- Contact Us

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Responsive Web Design
- Web Accessibility

## Features Included

- Working navigation bar on every page
- Responsive layout for desktop, tablet, and mobile screens
- Product and service cards with images and descriptions
- Product search/filter feature using JavaScript
- Contact form
- Blog/resource page with basic sustainability tips
- Table showing eco-friendly business practices
- Image alt text for accessibility
- Mobile-friendly design made using media queries

## How to Run the Project

1. Download or open the project folder from GitHub.
2. Open project folder.
3. Open `index.html` in a web browser.
4. Use the navigation bar located at the top to move between pages.
